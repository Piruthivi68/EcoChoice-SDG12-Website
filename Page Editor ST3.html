Challenges and Lessons Learned

Key challenges included:

  • Implementing the splash screen timer while maintaining accessibility
  • Creating responsive grids that work across devices
  • Ensuring consistent styling across all pages

Lessons learned:

  • Importance of planning CSS architecture early
  • Value of progressive enhancement for animations
  • Need for thorough cross-browser testing

Compliance

The pages comply with web standards by:

  • Using semantic HTML5 elements
  • Following WCAG 2.1 accessibility guidelines
  • Validating against W3C standards
  • Implementing responsive design principles
